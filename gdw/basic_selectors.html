Basic CSS Selectors

Element Selectors
Are used to style a specific element.

Class Selectros
Are used to project specific aspects of a certain element.

ID Selectors
Is used to take over a specific class style.

Group Selectors
Are used to add style to a collective group of selectors before the style.

Pseudo-class Selectors for Hyperlink
Are used to add style to regular links, clicked links, or when someone hovers over a link.
