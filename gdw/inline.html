Inline Elements

This is a strong element. This element portrays information that the writer wants the reader to catch their attention.

This is a bolded element. This element depicts important information in text.

This is an encoded element. This element emphasises a word.

This is an italicized element. This element italicizes a word.

This is a quotation element. This element puts quotation marks around a word.

This is an source code enclosed element. This element puts code into the html page.

This is an keyboard encolsed element. This element changes the text to look like a computer keyboard character.

This is an enclosed above subscript element. This element places smaler text above the rest of the text to work as a note to refer the reader to some other part of the text.

This is an enclosed below subscript element. This element places smaller text below the rest of the text to work as another way to refer the reader to another part of the text.

This is an incline logical span divisiont element. This element is used to group a few elements together in an organized format.
